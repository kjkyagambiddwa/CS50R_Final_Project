# centralMoments: Statistical Computing in R

## Project Overview
***centralMoments*** is an R package designed to automate the calculation of fundamental statistical measures. Developed as a final project for Harvard's CS50 introduction to programming with R, this tool computes central moments and key distributional statistics, returning results precisely to four decimal places.
This project showcases the transition from theoretical statistics to scalable software development.

## Key Features & Exports
The package provides specialized functions for rigorous data analysis:
- **Arithmetic Mean (my_mean):** Calculates the average value of a dataset.
- **General Moments (moment):** Computes the kth moment about the mean.
- **Variance (my_variance):** Measures data dispersion.
- **Skewness (my_skewness):** Quantifies the asymmetry of the probability distribution.
- **Kurtosis (my_kurtosis):** Measures the "tailedness" of the distribution.
